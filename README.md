# SoapsTeleporter Wiki

Welcome to the official wiki for **SoapsTeleporter**, a portal system built for Paper servers running Minecraft 1.21+.

---

## Version 1.0.1 Highlights

- Layered render pipeline with adaptive quality and visibility culling
- Optional condition expression support (`AND` / `OR` / `NOT`) per portal
- Multi-destination routing support (`fixed`, `random`, `weighted`, `permission`)
- Shape preview tools for build workflows (`/st shapepreview ...`)
- New runtime overlay metrics in `/st debug overlay`

---

## What is SoapsTeleporter?

SoapsTeleporter lets you create portals anywhere in your world. Walk into one portal, come out at another. You can set up particle effects, sounds, costs, conditions and more for each portal individually.
